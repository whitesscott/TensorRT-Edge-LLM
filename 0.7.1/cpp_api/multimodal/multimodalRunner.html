Multimodal Runner#

class MultimodalRunner#

Base class for multimodal vision-language model runners.

Provides interface for vision encoder processing in VLMs. Subclasses implement specific VLM architectures (Qwen-VL, InternVL, etc.).

Subclassed by trt_edgellm::rt::InternViTRunner, trt_edgellm::rt::NemotronOmniViTRunner, trt_edgellm::rt::Phi4MMViTRunner, trt_edgellm::rt::Qwen3OmniAudioRunner, trt_edgellm::rt::QwenViTRunner
